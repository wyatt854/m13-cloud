    [ CONSOLE COMMANDS ]

    M13S : Accès racine du Cloud distant.

    LOCAL : Ouvre le module d'encodage pour déposer un fichier de ton appareil vers la mémoire du navigateur.

    LSEE : Affiche tes fichiers stockés localement.

    [CODE PERSO] : Tape n'importe quel autre nom de dossier pour accéder aux sous-répertoires sécurisés.

    Note : Le stockage local est limité à ~5Mo. Les fichiers sont convertis en Base64.
